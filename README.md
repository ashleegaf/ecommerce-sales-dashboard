## Getting Started

**Create .env file variables:**

```bash
MONGODB_URI="_insert URI to mongodb database here_"
```

**Install packages:**

```bash
npm install
```

**Run the development server:**

```bash
npm run dev
```

**Seed the database:**
```bash
npm run seed
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.
